## Implementation notes
- Use `URL.createObjectURL(new Blob([csv], { type: 'text/csv' }))` + a hidden `<a>` element to trigger browser download — works offline, no server needed
- Column order: #, Name, G, PA, AB, H, 2B, 3B, HR, R, RBI, BB, SO, SB, AVG, OBP, SLG, OPS, IP, ERA, WHIP
- If a player has no pitching stats, leave IP/ERA/WHIP blank
- Format AVG/OBP/SLG/OPS as 3-decimal strings (e.g. ".312"); IP as "2.1" notation
- Filename: `{teamName}_{seasonName}_stats.csv` with spaces replaced by underscores
- Button only appears when `playerStats` is loaded and non-empty
